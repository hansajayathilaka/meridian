`cargo test -q -p meridian-cli --bin meridian opacity::tests::opacity_audit_passes -- --exact` — under
`set -euo pipefail` with no `|| true`, so the script's exit code is the test's real pass/fail. The test
(`apps/cli/src/opacity.rs`) builds a real Alice/Bob X3DH+ratchet session, seals real plaintext content,
accumulates the server-visible transcript, and asserts zero leaks across >100 envelopes.
