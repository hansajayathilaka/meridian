# 0.1 — Identity & Keystore Core

**Phase:** 0 — Foundation · **Feature:** T01 · **Depends on:** — · **Status:** done

## Goal
Mint, parse, and verify `mrd1:` identities (+ QR), with keys held in an encrypted-at-rest `SecretStore`.

## Scope
- **In:** identity encoding/verification, QR round-trip, keystore backends, conformance vectors.
- **Out:** trust/verification UX (T08), multi-device (T13).

## Deliverables
1. `apps/identity/` (`id.rs`, `keys.rs`, `qr.rs`), `apps/store/` `SecretStore`.
2. `apps/cli` mint/parse/verify commands.
3. Conformance vectors `test-vectors/identity-v1.json` + `apps/identity/tests/`.

## Risks / notes
Wire-critical crates (ed25519-dalek, data-encoding, crc32c) pinned; any bump must re-pass the vectors.

## Tests
`cargo nextest run -p meridian-identity` · `cargo run -p xtask -- vectors`

## Reviews
security-reviewer (key hygiene), architect (wire format), test-engineer (conformance).
